# Spam Mail Detection System

A Machine Learning and NLP based web application that classifies messages as Spam or Ham using TF-IDF Vectorization and Logistic Regression.

Built with Flask for real-time prediction through a simple web interface.

---

## 🚀 Features

- Spam/Ham message classification
- NLP based text processing
- TF-IDF feature extraction
- Logistic Regression model
- Real-time prediction using Flask

---

## 🧠 Machine Learning Workflow

1. Data Collection  
2. Data Preprocessing  
3. TF-IDF Vectorization  
4. Model Training  
5. Model Evaluation  
6. Flask Deployment  

---

## 📂 Dataset

| Column | Description |
|---|---|
| Category | Spam or Ham label |
| Message | Email/SMS text |

---

## 🤖 Model Used

- Logistic Regression

### Why Logistic Regression?
- Fast and lightweight
- Effective for text classification
- Easy to deploy

---

## 📈 Evaluation Metrics

The model was evaluated using:
